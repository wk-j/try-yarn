## Try Yarn

FAST, RELIABLE, AND SECURE DEPENDENCY MANAGEMENT - https://yarnpkg.com

## Init, Add, Run

```bash
yarn init
yarn add react
yarn add react-dom
yarn add @types/react
yarn add @types/react-dom
yarn run date
```
